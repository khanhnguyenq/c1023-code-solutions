# react-props-and-expressions-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What are props in React?
  information that you pass to a JSX tag, mainly these are CSS attributes

- How do you declare and access props in a component?
  declare props in the defining process of creating the component
  access props when you call the function and passing it a value for the prop

- How do you pass props to a component?
  by calling that component passing in the values for the defined parameters

- How do you write JavaScript expressions in JSX?
  by wrapping it in curly braces
